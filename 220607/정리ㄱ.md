함수 선언문
원래 너희가 쓰던 방식
```js
function 원래쓰던방식(){
    console.log("ㅎㅇ")
}
```

함수 표현식
함수를 중괄호로 표현 `{}`해서 사용할 수 있다
```js
const 간지나게함수쓰기 = function(){
    console.log("ㅎㅇ")
}
```

생성자 함수
불편하게 `new` 를 사용해서 함수를 만듬
```js
const 불편하게함수생성 = new Function("인자","실행코드")
```

# 동기/ 비동기
동기 = 코드가 하나하나 실행 = 내가 직접 하나하나
비동기 = 코드가 알아서 실행실행 = 딴일 하는거
